Session storage

Session storage stores data only till tab/window is closed

Print all variables in session storage
for(var i=0; i<sessionStorage.length; i++) {
  • console.log(sessionStorage.key(i), " = ", sessionStorage.getItem(sessionStorage.key(i)));
}

Local storage

Local storage is persistent storage
Browser has one storage for one domain (multiple apps on same domain can share same storage)
Accessed via javascript setItem() and getItem()

Print all variables in local storage
for(var i=0; i<localStorage.length; i++) {
  • console.log(localStorage.key(i), " = ", localStorage.getItem(localStorage.key(i)));
}
