Testing for Weak SSL/TLS Ciphers/Protocols/Keys vulnerabilities

nmap

Discover services wrapped by SSL
nmap -sV --reason -PN -n --top-ports 100 www.example. com

Checking for Certificate information, Weak Ciphers and SSLv2
nmap --script ssl-cert,ssl-enum-ciphers -p 443,465,993,995 www.example.com

Check for POODLE, HEARTBLEED vulnerability
nmap -sV --version-light --script ssl-poodle,ssl-heartbleed -p 443 <host>

openssl

Checking for Client-initiated Renegotiation and Secure Renegotiation
openssl s_client -connect www2.example.com:443
HEAD / HTTP/1.1
R
<CRLF>

If we get valid servers response then client renegotiation is enabled

TestSSLServer.jar

Testing supported Cipher Suites, BEAST and CRIME attacks
