GET BYTECODE
1. Download browser extension bytecode (check HTML and scripts for location)
2. Unpack bytecode by just renaming extension to .zip
  • .jar - Java package
  • .xap - Silverlight package
  • .swf - Flash (don't need extraction)
  • .class - Java bytecode
  • .dll - Silverlight bytecode

3. Decompile
  • www.varaneckas.com/jad - Java
  • https://github.com/java-decompiler/jd-gui
  • http://www.nowrap.de/flasm - Flash
  • http://www.nowrap.de/flare - Flash
  • www.hp.com/go/swfscan - Flash
  • www.red-gate.com/products/dotnet-development/reflector/ - Silverlight

REVIEW AND MODIFY SOURCE CODE

4. General code review
  • Does code contains public methods callable from outside script?
  • Can these methods be used to malicious purposes?
  • Intercept response and add javascript which can call these public methods.
5. Review input validation and output obfuscation and encryption
6. Hidden functionality which can be unlocked by modifying code
7. References to server-side functionality
8. Modify source code if there are no public methods which can be exploited from outside script

RECOMPILE AND PACK

9. Recompile with
  • javac - Java
  • flasm - Flash
  • Visual Studio - Silverlight

10. Pack back to .zip and rename extension properly

LOAD EXTENSION TO BROWSER
  • find extension file cached by browser on disk and replace
  • using intercepting proxy change url from wich extension is loaded
  • Request original URL, intercept response with original bytecode, change original bytecode with malicious (Burp: Paste from file)
