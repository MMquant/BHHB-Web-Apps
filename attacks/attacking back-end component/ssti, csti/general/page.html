Server Side Template Injection (SSTI)
Client Side Template Injection (CSTI)

Description

If backend server utilizes any templating engine then it can be vulnerable to template injection.
Injection may occur directly in browser if web app uses a client-side templating engine.

Server-side templating engines

Jinja2
Rails ERB
Liquid Engine

Client-side templating engines

AngularJS

Detection

Check specific template syntax and test simple injections such {{1+1}}
Test for RCE or JavaScript execution.
