Amass
https://github.com/OWASP/Amass

DNS Enumeration and Network Mapping


Switches

-active Attempt zone transfers and certificate name grabs
-d Domain
-o Path to the text output file
-rf Path to a file providing preferred DNS resolvers


Examples

service snapd start
amass -active -d example.com -o dns_amass_subdomains.txt -rf trusted_resolvers.txt


Notes

Update all programs installed by snap
snap refresh
