massdns
https://github.com/blechschmidt/massdns

Tool for bruteforcing DNS


Switches

-r Text file containing DNS resolvers
-w Write to the specified output file instead of standard output


Examples

./subbrute.py subdomains_all_unique_filtered.txt test-domain.com | massdns -r resolvers.txt -t A -w massdns_output.txt
