DNSweeper
https://github.com/MMquant/DNSweeper

Asynchronous public DNS auditing tool


Switches for enumerate command

-f Path to file with (scraped) subdomains
OR
-d Domain
-p Path to file with bruteforce payload
-r Reverse lookup regex matching pattern
--use-cache Use cached resolvers
--bruteforce-recursive Enable recursive bruteforce and use custom payload file
--exclude File with subdomains which not to enumerate
-v Verbosity -vv -vvv


Examples

python3.7 DNSweeper.py enumerate -d test-domain.com -p subdomains_all_unique_filtered.txt -r "test-domain\.com$" --bruteforce-recursive subdomains-10000.txt_ascii --exclude subdomains.txt
