dnsrecon
https://github.com/darkoperator/dnsrecon

DNS Enumeration Script


Examples

Perform search for all types of DNS records

dnsrecon -d test-domain.com -t std

Perform google search for subdomains

dnsrecon -d test-domain.com -t goo

Perform bing search for subdomains

dnsrecon -d test-domain.com -t bing

DNS cache snooping

dnsrecon -d test-domain.com -t snoop
