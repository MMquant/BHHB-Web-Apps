subfinder
https://github.com/subfinder/subfinder

SubFinder is a subdomain discovery tool


Switches

-d Domain
-b Bruteforce mode
-w Bruteforce wordlist
-t Threads
-dL List of input domains
-o Output file
-recursive Use recursion to find subdomains
-rL Text file containing list of resolvers to use
--no-passive Don't perform passive enumeration
--silent Show only subdomains found
-v Verbose


Examples

subfinder -d flickr.com -o dns_subfinder_subdomains.txt -rL trusted_resolvers.txt


Notes

subfinder uses following sites
  • Virustotal
  • Passivetotal
  • SecurityTrails
  • Censys
  • Riddler
  • Shodan

plus passive enumeration techniques

Upgrade subfinder before use

go get -u github.com/subfinder/subfinder
