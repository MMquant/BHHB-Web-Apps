host
https://linux.die.net/man/1/host

DNS lookup utility


Examples

Name servers

host -t ns <domain>

Mail servers

host -t mx <domain>

Brute force DNS forward lookup

for ip in $(cat list.txt);do host $ip.<domain.com> | grep "has address" | cut -d" " -f1,4;done

Brute force DNS reverse lookup (use IP range derived from DNS forward lookup)

for ip in $(seq 76 91);do host 38.100.193.$ip;done |grep -v "not found"

Zone transfers (list all internal and external subdomain if DNS badly configured)

host -l <domain> <dns server address>
