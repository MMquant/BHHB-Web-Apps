fierce
https://github.com/mschwager/fierce

A DNS reconnaissance tool for locating non-contiguous IP space


Examples

Perform complete DNS scan

fierce -dns zonetransfer.me -threads 3

DNS subdomains custom list bruteforce
I recommend to bruteforce with another tool as fierce is not verbose and you don't see progress state.

fierce -dns zonetransfer.me -wordlist /path/to/payload.txt
