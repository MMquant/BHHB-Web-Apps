Shodan
https://www.shodan.io/

IOT search engine


Vulnerable web components

This engine searches for given string throughout server responses and service banners. If we find a vulnerable web app, web server or other web component we can search shodan for machines running this vulnerable component.

Search Operators
  • title: Search the content scraped from the title HTML tag
  • html: Search the full HTML content of the returned page
  • product: Search the name of the software or product identified in the banner
  • net: Search a given netblock (example: 204.51.94.79/18)
  • version: Search the version of the product
  • port: Search for a specific port or ports
  • os: Search for a specific operating system name
  • country: Search for results in a given country (2-letter code)
  • city: Search for results in a given city

Negation is done with "!" operator (ie. title:"Outlook Web Access" !port:443,80)
