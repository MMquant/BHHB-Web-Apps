Detection
  • Submit arbitrary parameter name with clear malicious payload and check if request is blocked
  • Find reflections and test all ASCII characters if any is not returned or is modified

WAF bypass
  • Use benign strings for payloads
  • Try to resubmit malicious parameter in various inputs
    • GET request URL param
    • POST request body param,
    • POST request URL param
  • On ASP.NET resubmit malicious param as cookie name=value
