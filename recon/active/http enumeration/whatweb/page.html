whatweb
https://github.com/urbanadventurer/WhatWeb

Web application fingerprinting tool


Switches

-a=LEVEL
1. Stealthy Makes one HTTP request per target and also follows redirects.
3. Aggressive If a level 1 plugin is matched, additional requests will be made.
4. Heavy Makes a lot of HTTP requests per target. URLs from all plugins are attempted.

-U=AGENT User-agent
--follow-redirect=WHEN never, http-only, meta-only, same-site, same-domain, always (default)
-c=COOKIES name=value; name2=value2
--proxy <hostname[:port]>
-v verbose
-t threads
--wait=SECONDS Wait SECONDS between connections

plugins
-l List plugins
--info-plugins <SEARCH> Detailed info about plugin


Examples

whatweb -a 3 www.wired.com
whatweb -a 3 https://67.195.205.33:80
