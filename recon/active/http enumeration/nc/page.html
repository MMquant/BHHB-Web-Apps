Netcat
https://en.wikipedia.org/wiki/Netcat

TCP/UDP networking tool


Examples

Fingerprinting webserver

nc <IP> <port>
HEAD / HTTP/1.1
<CRLF>

Discover active HTTP servers

nc <IP> <port>
GET / HTTP/1.1
Host: www.victim.com
<CRLF>

Discover accepted HTTP methods

nc <IP> <port>
OPTIONS / HTTP/1.1
Host: www.victim.com
<CRLF>
