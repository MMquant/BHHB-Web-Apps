httprint
http://www.net-square.com/httprint.html

web server fingerprinting tool


Switches

-h host
-i input file with hosts
-s file with signatures
-P0 no ping
-nr threads (default 8)


Examples

Download fresh signatures before use

curl -o /root/Programs/httprint_301/linux/signatures.txt http://net-square.com/signature.txt

Fingerprint target webserver

httprint -h https://www.cez.cz/ -s /root/Programs/httprint_301/linux/signatures.txt

Or use directly with IP

host www.cez.cz
89.111.76.144

httprint -P0 -h 89.111.76.144 -s /root/Programs/httprint_301/linux/signatures.txt
