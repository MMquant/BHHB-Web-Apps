eyewitness
https://github.com/FortyNorthSecurity/EyeWitness

Mass banner grabber


Use for inspecting HTTP ports in large IP ranges


Switches

-web HTTP Screenshot using Selenium
-f Filename Line seperated file containing URLs to capture
--prepend-https Prepend http:// and https:// to URLs without either
--jitter Randomize URLs and add a random delay between requests in seconds
--all-protocols


Example

./EyeWitness.py --web --prepend-https --all-protocols-f subdomains_unique.txt
