Masscan
https://github.com/robertdavidgraham/masscan

Tool for mass port scanning written in C


Switches

-p Ports (TCP)
-pU Ports (UDP)
--range IP or CIDR range
--resume Resume scan if CTRL-C pressed before
--http-user-agent Custom user agent
--open-only Report only open ports
--shards <x>/<y> Split scan among y instances. x identifies current shard id
--output-format xml, binary, grepable, list, or JSON
--output-filename Output filename
--rate 100000 Maximum packets per seconds
-oG Sets the output format to grepable and saves the output in the given filename
-oJ Sets the output format to JSON and saves the output in the given filename
-oL Sets the output format to a simple list format and saves the output in the given filename


Examples

Sharding scan

masscan 0.0.0.0/0 -p0-65535 --shard 1/3
masscan 0.0.0.0/0 -p0-65535 --shard 2/3
masscan 0.0.0.0/0 -p0-65535 --shard 3/3

Scan IPs from file (TCP)

for ip in $(cat unique_ips_sorted.txt); do echo $ip; masscan $ip -p0-65535 ; done

Scan IPs from file (UDP)

for ip in $(cat unique_ips_sorted.txt); do echo $ip; masscan $ip --ports U:0-65535 ; done
